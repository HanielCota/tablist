# Security Policy

## Supported versions

The latest released version of Tablist receives security fixes. Older versions are not
maintained — please update before reporting.

## Reporting a vulnerability

**Please do not open a public issue for security problems.**

Instead, report privately via GitHub's
[security advisories](https://github.com/HanielCota/tablist/security/advisories/new), or email
**pessoal@hanielfialho.com**.

Please include:

- A description of the vulnerability and its impact.
- Steps to reproduce (config, commands, server version).
- Any suggested fix, if you have one.

You can expect an initial response within a few days. Once the issue is confirmed and fixed, a
patched release will be published and the reporter credited (unless anonymity is requested).
